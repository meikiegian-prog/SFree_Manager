# SFree_Manager

#### 介绍
智管自由（SFree_Manager）是一款专注于个人效率管理的微信小程序，旨在帮助用户便捷地管理日常任务、规划时间、记录进度，并通过数据统计提供个性化的效率分析，让时间管理更简单、自由。


#### 软件架构
- 前端：基于微信小程序原生框架开发，使用WXML（页面结构）、WXSS（样式）、JavaScript（逻辑）构建用户界面及交互
- 数据存储：支持微信云开发数据库（实时同步）与本地缓存（离线数据暂存）
- 后端：依托微信云函数实现核心业务逻辑（如数据处理、权限验证等）
- 第三方支持：集成微信登录、微信支付（可选，用于增值服务）


#### 安装教程

1.  克隆/下载本仓库代码至本地：`git clone [仓库地址]`
2.  打开微信开发者工具，选择「导入项目」，填写小程序AppID（测试可选择「测试号」）
3.  等待依赖加载完成后，点击「编译」即可在模拟器中运行；若需真机调试，连接设备后点击「预览」生成二维码，用微信扫描即可


#### 使用说明

1.  **注册登录**：打开小程序后，通过微信一键授权登录，无需额外注册
2.  **任务管理**：
    - 在「任务列表」页面点击「+」添加新任务，设置标题、截止时间、优先级
    - 滑动任务项可进行「标记完成」「编辑」「删除」操作
3.  **数据统计**：在「统计」页面查看每日/每周/每月任务完成情况、效率趋势图表
4.  **设置中心**：可修改个人信息、调整提醒设置（如任务到期提醒）


#### 参与贡献

1.  Fork 本仓库到个人账号
2.  基于 `dev` 分支新建功能分支，命名格式：`Feat_功能名称`（如 `Feat_添加日历视图`）
3.  提交代码时需遵循代码规范：
    - 变量/函数命名使用小驼峰式（如 `taskList`）
    - 提交信息格式：`[类型] 描述`（如 `[Feat] 新增任务搜索功能`）
4.  完成开发后，向原仓库 `dev` 分支提交 Pull Request，并备注功能说明及测试情况


#### 特色功能

1.  极简界面：无冗余信息，专注任务本身，降低操作干扰
2.  多端同步：微信账号登录，数据自动同步至云端，换设备不丢失
3.  智能提醒：支持自定义时间提醒，避免遗漏重要任务
4.  数据可视化：通过图表直观展示效率变化，助力优化时间分配
